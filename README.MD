# Финансовый бот для двоих 🤖💑

Телеграм-бот для совместного учета финансов, планирования и отслеживания расходов.

## 📋 Функционал
- 📊 Учет расходов и доходов
- 👫 Общая статистика для двоих
- 📅 Ежедневник с напоминаниями
- 🛒 Список планируемых покупок
- 📈 Анализ и сравнение финансов

## 🚀 Установка
1. Клонируйте репозиторий
2. Установите зависимости: pip install -r requirements.txt
3. Создайте файл .env с переменными окружения
4. Запустите бота: python bot.py
